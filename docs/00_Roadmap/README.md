# 1. Propósito

Este documento constituye el Roadmap Maestro del proyecto MUNO y define la estrategia general para llevar la plataforma desde su estado actual hasta una versión estable, segura, documentada y preparada para su implementación en municipios.

El Roadmap funciona como documento rector del Expediente Técnico y sirve como referencia para todas las etapas del Hito 2.

---

# 2. Objetivo General

Desarrollar una versión completamente estable de MUNO manteniendo todas las funcionalidades existentes, corrigiendo errores de lógica, arquitectura, seguridad y rendimiento, sin incorporar nuevas funcionalidades durante el proceso de estabilización.

El objetivo final es disponer de una plataforma apta para iniciar un piloto municipal y evolucionar posteriormente hacia una solución SaaS escalable para múltiples municipios.

# 4. Objetivos del Hito 2

Durante esta etapa se buscará:

- Consolidar todas las auditorías realizadas.
- Construir el Documento Maestro de Hallazgos.
- Verificar cada hallazgo mediante una auditoría independiente.
- Elaborar el Plan Maestro de Corrección.
- Corregir todos los problemas detectados.
- Validar estabilidad.
- Validar seguridad.
- Validar funcionamiento.
- Preparar la aplicación para pruebas piloto.

No se incorporarán nuevas funcionalidades durante este proceso.

---

# 5. Principios del Proyecto

Durante todo el Hito 2 deberán respetarse los siguientes principios:

- Mantener todas las funcionalidades existentes.
- No eliminar módulos funcionales.
- Priorizar estabilidad sobre nuevas características.
- Priorizar seguridad.
- Priorizar mantenibilidad.
- Mantener documentación actualizada.
- Mantener trazabilidad completa mediante Git.
